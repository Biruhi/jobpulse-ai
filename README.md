# 💼 JobPulse AI

🚀 **Live Demo**
https://jobpulse-ai-sy9d6njvbmh5xcmcbojx7n.streamlit.app/

JobPulse AI is a job intelligence and hiring analytics platform built with Python and Streamlit. It searches remote job listings, analyzes hiring trends, identifies in-demand skills, and generates business-ready analytics dashboards.

---

## 🚀 Features

### 🔍 Job Search

Search jobs by keyword:

* Python Developer
* Data Analyst
* Machine Learning Engineer
* Software Engineer
* Teacher

---

### 📊 Executive Dashboard

Key metrics include:

* Jobs Found
* Companies Hiring
* Locations
* Hiring Distribution

---

### 🛠 Skill Analysis

Automatically identifies:

* Most Requested Skills
* Technology Trends
* Hiring Demand

Examples:

* Python
* SQL
* AWS
* Docker
* Machine Learning

---

### 🌍 Location Analytics

Analyze:

* Jobs by Location
* Remote Opportunities
* Geographic Hiring Trends

---

### 💼 Job Details

View:

* Job Title
* Company
* Location
* Date Posted
* Skills
* Application Link

---

### 📤 Export Reports

Export results to:

* Excel (XLSX)

## ▶️ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run dashboard/streamlit_app.py
```
